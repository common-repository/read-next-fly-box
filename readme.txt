This plugin will create a read next post box, the data is fetched from the Wordpress database randomly. The objective is to decrease the bounce rate of your site and improve user experience.

== Description ==


<strong>What This Plugin Does?</strong>

This plugin searchs for your posts and randomly choses one or two to be shown in a related box. This box is initially hidden and appears after the user scrolls down the page to a certain point.

That all mean lower user bounce rate and better user experience.

== Installation ==


1. Upload `read-next-fly-box` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the Options of the plugin (App Id, App Secret and Admin E-mail are mandatory)
4. Profit :)
